# ReverseMind — Reflex Challenge

ReverseMind คือเกมฝึกปฏิกิริยาและสมาธิ ที่ออกแบบมาเพื่อทดสอบความเร็วในการตัดสินใจ และความสามารถในการ “คิดสวนสัญชาตญาณ”

กติกาง่ายมาก:

> กดปุ่ม “ตรงข้าม” กับลูกศรที่แสดงบนหน้าจอ ก่อนเวลาจะหมด

เมื่อคะแนนเพิ่มขึ้น เกมจะค่อย ๆ เร็วขึ้น ทำให้ผู้เล่นต้องใช้ทั้งสมาธิ ความแม่นยำ และการตอบสนองที่รวดเร็วมากขึ้นเรื่อย ๆ

---

## ฟีเจอร์ภายในเกม

* Countdown ก่อนเริ่มเกม
* ระบบเพิ่มความยากอัตโนมัติ
* ระบบจับเวลาแบบ Real-time
* ระบบ Accuracy (% ความแม่นยำ)
* ระบบ Survival Time
* ระบบบันทึก Best Score ด้วย localStorage
* UI สไตล์ Minimal Futuristic
* รองรับการควบคุมด้วย Keyboard

---

| ลูกศรที่แสดง | ปุ่มที่ต้องกด |
| ------------ | ------------- |
| ⬅            | ➡             |
| ➡            | ⬅             |
| ⬆            | ⬇             |
| ⬇            | ⬆             |

เกมจะเร็วขึ้นตามคะแนนที่ทำได้

## สถิติภายในเกม

หลังจบเกม ระบบจะแสดง:

* คะแนนปัจจุบัน (Score)
* คะแนนสูงสุด (Best Score)
* ความแม่นยำ (Accuracy)
* เวลาที่อยู่รอด (Survival Time)

## วิธีติดตั้ง

Clone โปรเจกต์:

```bash
git clone https://github.com/your-username/reversemind.git
```

ติดตั้ง dependencies:

```bash
npm install
```

เริ่มรันโปรเจกต์:

```bash
npm run dev
```

Build สำหรับ Production:

```bash
npm run build
```

## แนวคิดของเกม

ReverseMind ได้รับแรงบันดาลใจจากเกมฝึกสมองและเกม Reflex Training โดยเน้นการฝึก:

* ความเร็วในการตอบสนอง
* สมาธิ
* การควบคุมสัญชาตญาณ
* การตัดสินใจภายใต้แรงกดดัน
